# GitFront.io Sharable Link Setup

This PUBLIC repository has a PUBLICLY sharable [readonly] link for sharing without requiring a GitHub account.

## Overview

GitFront.io allows you to create secure, sharable links to repositories without exposing your GitHub credentials or repository settings.

## Repository Access

**GitFront Sharable Link:**
```
https://gitfront.io/r/dynotesting/wKMQyrmpJYXe/stable-releases/
```

This link provides read-only access to the stable-releases repository for authorized users.

## How It Works

1. Original repository is hosted on GitHub (public)
2. GitFront.io mirrors the repository with a unique sharable URL
3. Users can access the repository contents without GitHub credentials
4. Access control is managed through the unique token in the GitFront URL

## Usage

To access the repository contents, navigate to the GitFront link provided above. You can:
- Browse the file structure
- View file contents
- Clone using the provided GitFront URL

## Notes

- This is a read-only mirror for sharing purposes
- The primary repository remains on GitHub
- Changes should be made to the primary GitHub repository
